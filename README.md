🚀 PRODDEC CEC – Innovation Universe

A modern, animated, and visually immersive website for PRODDEC (Product Development & Innovation Cell) at College of Engineering Chengannur (CEC), Kerala.

Built using Vanilla HTML, CSS, and JavaScript, featuring smooth animations, starfield canvas effects, and a bold black-accent design system.

✨ Features
🎨 Design Highlights

🌌 Animated Starfield Hero – Canvas-powered space animation with mouse interaction
🌑 Dark Premium Theme – Black-based UI with gold/orange accents
✨ Smooth Scroll Effects – Clean transitions between sections
🎴 Animated Cards – Scroll-reveal animations using Intersection Observer
🧭 Interactive Navbar – Dynamic resize and shadow on scroll
📱 Fully Responsive – Optimized for mobile, tablet, and desktop
🎯 Modern Typography – Google Fonts (Poppins)

🌐 Website Sections

🏠 Hero Section – Animated cosmic background with call-to-action
ℹ️ About Section – Mission, Vision, Community
💼 Projects Section – Featured technical initiatives
🎉 Events Section – Hackathons, Workshops, Meetups
📩 Contact Section – Official contact details
🔻 Footer – Institutional credits
